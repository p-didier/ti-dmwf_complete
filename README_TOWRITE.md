## Getting Started

### Prerequisites

- Python 3.XX or higher
- Other dependencies listed in `requirements.txt`

### Installation

1. Clone the repository:
        ```sh
        git clone <<this repo's URL>>
        ```
2. Navigate to the project directory:
        ```sh
        cd <<the folder>>
        ```
3. Install the required packages (possibly in a virtual environment):
        ```sh
        pip install -r requirements.txt
        ```

### Usage

1. Set up desired configuration in the appropriate YAML file.
2. Run the main script in VS Code or using:
        ```
        python main.py
        ```
